Storage Selector

An Angular directive that provides the Storage Icon button to launch a Storage modal and select a file

Dependencies Angular JS Angular Bootstrap

Usage

When clicking the Storage icon button, the Storage modal is launched and you may need to allow Rise Vision access to your Google account to verify who you are. Once verified, you can select a file from the list.

The Storage Selector directive will $emit an event called "picked" if you click the "Select" button in the Storage modal and this event will provide an array of urls of the selected files.

Alternatively, if you prefer not to use the "picked" event or you're implementing a vanilla JS approach for storage selection, when a file is selected, the Storage modal will send a message to its parent window. If you click "Select" button in the Storage modal the message will be an array of the urls of the selected files. If you click "X" in the Storage modal then the message will be a string stating "close".

This directive provides attributes local and companyid. The local attribute tells the directive if it should use local files list. If the local attribute is not present than it is to use a company id to view it's storage bucket. To retrieve the Company ID click on Settings from the menu of the Rise Vision Application. The ID for the currently selected Company can be found in the URL where the ID is everything to the right of "company=", for example:

http://rva.risevision.com/#COMPANY_MANAGE/company=XXXXXXXX-XXXX-XXXX-XXXXXXXXXXXX

The directive also has attribute options instanceTemplate to point to a different template, and useCtrl to point to a different controller.

Module Includes

angular.module('myApp', ['risevision.widget.common.storage-selector']);

Script Includes

Make sure you include the following files in index.html

bower_components/angular/angular.js bower_components/angular-bootstrap/ui-bootstrap-tpls.js bower_components/component-storage-selector/dist/storage-selector.js
